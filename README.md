# Pulseboard

A static work pulseboard for visualizing and tracking productivity metrics.

## Features

- Real-time pulse data visualization
- Static site generation for easy deployment

## Installation

1. Clone the repository.
2. Install dependencies: `npm install`
3. Build the static site: `npm run build`

## Usage

Serve the built files using any static server.
